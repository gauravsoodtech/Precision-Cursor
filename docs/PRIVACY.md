# Privacy

DPI Assistant is a local Windows utility.

## Data Collection

DPI Assistant does not collect, store, upload, or sell user data.

## Network Access

DPI Assistant does not include telemetry, update checks, analytics, or network features.

## Local Input Handling

While enabled, the app observes mouse movement and keyboard hotkeys needed for its controls:

- Mouse movement is used to calculate the locked cursor position.
- `Ctrl+Alt+L` toggles line lock.

This input is handled locally in memory and is not written to files by the app.

## Logs

DPI Assistant does not create log files by default.

If future versions add diagnostics, logs should avoid credentials, tokens, private documents, and other sensitive information.
