# Changelog

All notable changes to Precision Cursor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project follows semantic versioning when releases are tagged.

## [Unreleased]

### Added

- Initial public Windows tray utility.
- Global `Ctrl+Alt+L` toggle hotkey.
- Global `Esc` emergency disable.
- 8-way cursor locking for horizontal, vertical, and diagonal movement.
- Raw input based direction detection.
- Direction smoothing for steadier lines.
- Tray menu with status, toggle, and exit actions.
- Unit tests for cursor snapping and locking behavior.
